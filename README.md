# Volta ao Mundo - Chile 🌄

Projeto acadêmico desenvolvido com HTML, CSS, Bootstrap e JavaScript.

## 📌 Objetivo
Site informativo sobre o Chile com páginas de turismo, cultura, gastronomia e história.

## 🚀 Tecnologias
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## 📂 Estrutura
- Home com hero section
- Navbar responsiva
- Footer multi-colunas
- Páginas internas
