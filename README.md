# 🌎 Volta ao Mundo - Chile

Projeto web desenvolvido para apresentar o Chile de forma visual e interativa, explorando turismo, cultura, gastronomia e geografia com foco em experiência do usuário e responsividade.

---

## 📌 Sobre o Projeto

O site tem como objetivo explorar diferentes aspectos do Chile, incluindo turismo, cultura, gastronomia, história e geografia.

A proposta foi criar uma experiência visual agradável, utilizando imagens marcantes e um layout moderno para destacar a diversidade do país.

---

## 🎯 Funcionalidades

* Navbar responsiva com menu hamburguer
* Carrossel interativo na página inicial
* Galeria de imagens com glightbox
* Seções organizadas por tema
* Cards de turismo com descrições
* Layout alternado (imagem + texto)
* Design responsivo para diferentes dispositivos

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5 (layout e componentes)
* JavaScript

---

## 🎨 Design

O projeto utiliza uma identidade visual baseada em cores escuras para valorizar as imagens e melhorar o contraste.

Também foram aplicados:

* overlays para legibilidade
* grid do Bootstrap para responsividade
* organização visual baseada em padrões modernos de UI, como uso de cards, overlays e grid responsivo

---

## 🌍 Estrutura do Site

* **Home** → Destaques principais com navegação visual
* **Turismo** → Pontos turísticos com detalhes
* **Cultura** → Tradições, arte e identidade chilena
* **Gastronomia** → Pratos típicos e descrições
* **História & Geografia** → Contexto histórico e características naturais

---

## 🧠 Desafios e Aprendizados

Durante o desenvolvimento, alguns desafios foram enfrentados, como:

* organização de arquivos e caminhos no GitHub Pages
* responsividade em diferentes telas
* integração entre HTML, CSS e JavaScript

Esses pontos contribuíram para um melhor entendimento sobre estrutura de projetos web e boas práticas de desenvolvimento.
